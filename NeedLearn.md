## ssh key
```
ls ~/.ssh 
#ls your ssh key

ssh-keygen
#create a ssh key

cat ~/.ssh/xxx.pub | pbcopy
#copy your public key
`pbcopy` // mac's keyboard shortcut, can let user copy what they want to copy
```

所以brew link ruby如果還行就要
brew link --overwrite ruby
